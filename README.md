# 🌸 REN Bot — Инструкция по запуску

## Что нужно перед началом
- Аккаунт в Telegram
- Аккаунт на GitHub (бесплатно)
- Аккаунт на Railway (бесплатно, через GitHub)
- Аккаунт на console.anthropic.com (для Claude API)

---

## Шаг 1 — Создать бота в Telegram (2 минуты)

1. Открой Telegram, найди @BotFather
2. Напиши `/newbot`
3. Придумай название: `REN Studio`
4. Придумай username: `ren_flowers_bot` (должен заканчиваться на bot)
5. Скопируй токен — длинная строка вида `7123456789:AAH...`
6. Сохрани его — понадобится на шаге 4

---

## Шаг 2 — Узнать свой Telegram ID

1. Найди в Telegram бота @userinfobot
2. Напиши ему `/start`
3. Он ответит твой ID — число вида `123456789`
4. Сохрани его

---

## Шаг 3 — Получить Claude API ключ (5 минут)

1. Перейди на https://console.anthropic.com
2. Зарегистрируйся (можно через Google)
3. Раздел "Billing" → пополни на $10 (хватит на 2-3 месяца)
4. Раздел "API Keys" → "Create Key" → скопируй ключ `sk-ant-...`
5. Сохрани его

---

## Шаг 4 — Загрузить код на GitHub

1. Зайди на https://github.com
2. Нажми "New repository" → название `ren-bot` → Create
3. Загрузи все файлы бота в репозиторий
   (кнопка "uploading an existing file" на странице репозитория)

---

## Шаг 5 — Деплой на Railway (10 минут)

1. Перейди на https://railway.app
2. "Login with GitHub" → авторизуйся
3. "New Project" → "Deploy from GitHub repo"
4. Выбери репозиторий `ren-bot`
5. Railway начнёт деплой — подожди 1-2 минуты

### Добавить переменные окружения:
1. Кликни на проект → вкладка "Variables"
2. Добавь по одной:
   ```
   BOT_TOKEN = твой_токен_от_BotFather
   ANTHROPIC_API_KEY = твой_ключ_sk-ant-...
   DIRECTOR_ID = твой_telegram_id
   TIMEZONE = Europe/Moscow
   ```
3. Railway автоматически перезапустит бот

---

## Шаг 6 — Первый запуск (5 минут)

1. Открой Telegram, найди своего бота по username
2. Напиши `/start` — бот зарегистрирует тебя как директора
3. Отправь флористам ссылку на бота: `t.me/ren_flowers_bot`
4. Каждый флорист пишет `/start` → вводит своё имя

---

## Команды директора

| Команда | Что делает |
|---------|-----------|
| `/отчет` | Полный отчёт: KPI + продажи |
| `/kpi` | KPI всех флористов |
| `/продажи` | Продажи букетов по каналам |
| `/витрина` | Все активные букеты |
| `/настройки` | Изменить время, пороги, сроки |
| Любой текст | Вопрос Claude — анализ данных |

## Команды флориста

| Команда | Что делает |
|---------|-----------|
| `/мой_кпи` | Мой KPI за месяц |
| `/витрина` | Мои активные букеты |
| `/букет` | Добавить новый готовый букет |
| `/статус_5` | Изменить статус букета #5 |

---

## Если что-то сломалось

- Railway → вкладка "Logs" — там видны ошибки
- Напиши мне в Claude — помогу разобраться
- Бот автоматически перезапускается при сбоях

---

## Расписание бота

- **10:00** — запрос начала смены флористам
- **14:00** — задача витрины букетов
- **15:00** — задача Flowwow (каждые 2 дня)
- **18:00** — задача витрины композиций
- **Каждые 30 мин** — проверка просроченных задач
- **Каждый час** — проверка срока годности букетов
- **Каждый пн 09:05** — еженедельный анализ от Claude
- **1-е число 09:00** — автоматический месячный отчёт

Все времена можно изменить через `/настройки`.
